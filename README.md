# Tugas-D-41
Superstore Sales Analysis Dashboard adalah proyek Data Science untuk mentransformasi data transaksi US Superstore menjadi wawasan strategis. Menggunakan Python, Streamlit, dan Plotly, dashboard ini memantau KPI utama seperti Revenue, Profit Margin, dan tren pasar melalui visualisasi interaktif dan desain modern bergaya Glassmorphism.
